Making a Nodebot with Windows on Devices Posted on Jun 23, 2014

Wireless Nodebot The Intel Galileo with Windows is a great platform for building NodeBots as the board itself can run the node server that controls the robot. This allows you to use one board for your server and hardware. Windows on the Galileo makes arduino development easy and much more powerful since you are able to use Visual Studio and other tools that you already know. In the case of this project however you can do all of your arduino programming through javascript using Nodejs and Cylonjs or Johnny-Five.
